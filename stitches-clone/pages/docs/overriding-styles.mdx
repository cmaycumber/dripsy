---
title: Overriding Styles
description: How to override styles in the consumption layer.
---

Stitches provides a `css` prop for overriding styles easily.

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button
    css={{
      borderRadius: '0',
      ':hover': {
        backgroundColor: 'black',
        color: 'white',
      },
    }}
  >
    Button
  </Button>
);
```

You can use tokens in the `css` prop.

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button
    css={{
      borderRadius: '0',
      ':hover': {
        backgroundColor: '$blue500',
      },
    }}
  >
    Button
  </Button>
);
```

You can also use `breakpoints` in the `css` prop.

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button
    css={{
      borderRadius: '0',
      ':hover': {
        backgroundColor: '$blue500',
      },
      bp2: {
        backgroundColor: '$yellow500',
      },
      bp3: {
        backgroundColor: '$purple500',
      },
      bp4: {
        backgroundColor: '$red500',
      },
    }}
  >
    Button
  </Button>
);
```

#### Overriding the HTML tag

Stitches provides an `as` prop for changing which tag a component outputs.

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  display: 'inline-flex',
  alignItems: 'center',
  fontSize: '13px',
  fontFamily: '$untitled',
  textDecoration: 'none',
  lineHeight: '1',
  fontWeight: 500,
  border: '0',
  height: '$5',
  paddingLeft: '10px',
  paddingRight: '10px',
});

render(
  <Button as="a" href="#">
    Button
  </Button>
);
```
