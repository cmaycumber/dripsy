#### Object syntax

In order to keep the bundle size to a minimum, Stitches uses JavaScript object notation rather than string syntax. Below, are examples of all of the common use cases.

#### Pseudo-class

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // Pseudo-class selector
  ':hover': {
    transform: 'translateY(-2px)',
    boxShadow: '0 10px 25px rgba(0, 0, 0, .3)',
  },
});

render(<Button>Button</Button>);
```

#### Pseudo-element

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',
  position: 'relative',

  // Pseudo-element selector
  '::before': {
    content: `""`,
    display: 'block',
    backgroundImage: 'linear-gradient(to right, #1fa2ff, #12d8fa, #a6ffcb)',
    position: 'absolute',
    top: '-3px',
    left: '-3px',
    width: 'calc(100% + 6px)',
    height: 'calc(100% + 6px)',
    borderRadius: 'inherit',
    zIndex: -1,
  },
});

render(<Button>Button</Button>);
```

#### Descendant selector

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // Descendant selector
  svg: {
    display: 'inline-block',
    verticalAlign: 'bottom',
    height: '13px',
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button
    <svg viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </svg>
  </Button>
);
```

#### Class selector

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // Class selector
  '&.custom-class': {
    marginLeft: '10px',
  },
});

render(
  <>
    <Button>Button</Button>
    <Button className="custom-class">Button</Button>
  </>
);
```

> In addition to the class selector, you can style all other available basic selectors such as [ID selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/ID_selectors), [attribute selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Attribute_selectors), [universal selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Universal_selectors), and [type selectors](https://developer.mozilla.org/en-US/docs/Web/CSS/Type_selectors).

#### Combinator selectors

```jsx live manual
const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // Ajdacent sibling selector
  '& + button': {
    marginLeft: '10px',
  },

  // Child selector
  '& > button': {
    marginLeft: '10px',
  },
});

render(
  <>
    <Button>Button</Button>
    <Button>Button</Button>
  </>
);
```

#### Refer to another component

```jsx live manual
const Icon = styled('svg', {
  display: 'inline-block',
  verticalAlign: 'bottom',
  width: '16px',
});

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style another component
  [`& ${Icon}`]: {
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button
    <Icon viewBox="0 0 32 32" fill="none" stroke="currentcolor" strokeWidth="2">
      <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
    </Icon>
  </Button>
);
```

#### Style a React component

```jsx live manual
const RightArrow = () => (
  <svg
    className="right-arrow"
    viewBox="0 0 32 32"
    fill="none"
    stroke="currentcolor"
    strokeWidth="2"
  >
    <path d="M22 6 L30 16 22 26 M30 16 L2 16" />
  </svg>
);

// add a `toString` method
RightArrow.toString = () => '.right-arrow';

const Button = styled('button', {
  backgroundColor: 'gainsboro',
  borderRadius: '9999px',
  fontSize: '13px',
  lineHeight: '1',
  fontWeight: 500,
  paddingTop: '10px',
  paddingBottom: '10px',
  paddingLeft: '16px',
  paddingRight: '16px',
  border: '0',

  // style any react component
  [`& ${RightArrow}`]: {
    display: 'inline-block',
    verticalAlign: 'bottom',
    height: '13px',
    marginLeft: '6px',
  },
});

render(
  <Button>
    Button <RightArrow />
  </Button>
);
```

> Remember to add the `toString` method on your component. That needs to return a string matching the desired selector. The selector can be anything you want.

#### Numeric values

Just like with React, jQuery, and other libraries, Stitches converts numeric values to `px`, if the CSS property doesn't accept unitless values.

```jsx
fontSize: 13, // Outputs font-size: 13px
paddingTop: 10, // Outputs padding-top: 10px
lineHeight: 1, // Outputs line-height: 1
fontWeight: 500, // Outputs font-weight: 500
```

#### Shorthand CSS properties

You can use shorthand CSS properties if you prefer. Stitches will parse the shorthand and convert it to longhand automatically.

```jsx
margin: 10,
padding: '10px 20px 30px',
border: '1px solid coral',
transition: 'all 200ms ease',
```

> [Tokens](/docs/tokens) will also be parsed in shorthand properties.

#### Keyframe animations

```jsx
const scaleUp = css.keyframes({
  '0%': { transform: 'scale(1)' }
  '100%': { transform: 'scale(1.5)' }
});

const Button = styled('button', {
  ':hover': {
    animation: `${scaleUp} 200ms`
  }
});
```

#### Global styles

For handling things like global resets, you can write global CSS styles.

```jsx
css.global({
  body: { margin: '0' },
});
```
