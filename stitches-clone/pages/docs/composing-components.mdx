---
title: Composing Components
description: Composing one Stitches component into another.
---

Stitches components can be composed via the `styled` function.

```jsx live manual
const BaseButton = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '15px',
  lineHeight: '1',
  fontWeight: 500,
  height: '35px',
  paddingLeft: '15px',
  paddingRight: '15px',
  border: '0',
});

const PillButton = styled(BaseButton, {
  borderRadius: '9999px',
});

render(
  <div style={{ display: 'flex', gap: '20px' }}>
    <BaseButton>BaseButton</BaseButton>
    <PillButton>PillButton</PillButton>
  </div>
);
```

When you compose a component which has variants, they will be inherited.

```jsx live manual
const BaseButton = styled('button', {
  backgroundColor: 'royalblue',
  color: 'white',
  fontSize: '15px',
  lineHeight: '1',
  fontWeight: 500,
  border: '0',

  variants: {
    size: {
      small: {
        fontSize: '13px',
        height: '$5',
        paddingLeft: '10px',
        paddingRight: '10px',
      },
      large: {
        fontSize: '17px',
        height: '$7',
        paddingLeft: '20px',
        paddingRight: '20px',
      },
    },
  },
});

const PillButton = styled(BaseButton, {
  borderRadius: '9999px',
});

render(
  <div style={{ display: 'flex', gap: '20px', alignItems: 'center' }}>
    <BaseButton size="small">BaseButton</BaseButton>
    <PillButton size="small">PillButton</PillButton>
    <BaseButton size="large">BaseButton</BaseButton>
    <PillButton size="large">PillButton</PillButton>
  </div>
);
```
